# ipa_dictionary

Từ điển ngôn ngữ các dân tộc - Đơn vị E246

## Mô tả

Ứng dụng từ điển đa ngôn ngữ hỗ trợ dịch giữa tiếng Việt và các ngôn ngữ dân tộc thiểu số:
- Tiếng Tày
- Tiếng Dao
- Tiếng Mông
- Tiếng Nùng
- Tiếng Lô Lô
- Tiếng Sán chỉ
- Tiếng Sán chay
- Tiếng Sán Dìu

## Tính năng

- Tra cứu từ điển theo cặp ngôn ngữ
- Tìm kiếm từ vựng
- Thêm/sửa/xóa mục từ
- Phát âm (TTS hoặc audio)
- Ghi chú và ví dụ cho mỗi mục từ

## Cài đặt

```bash
flutter pub get
flutter run
```
